# Personal Recipe Database

A small web app that suggests recipes based on the ingredients you already have.

## Idea

Instead of searching for recipes first, users enter the ingredients in their pantry and the app suggests recipes they can cook.

## Features (planned)

* Ingredient-based recipe search
* Recipe database stored in JSON
* Match percentage between pantry and recipe ingredients
* Display missing ingredients

## Tech Stack

* HTML
* CSS
* JavaScript

## Project Structure

```
index.html      → UI
styles.css      → styling
database.js     → recipe data
script.js       → matching logic
```
